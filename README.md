## Exploratory Data Analysis

This project presents an Exploratory Data Analysis (EDA) of the Online Retail II dataset using Python, Pandas, NumPy, and Matplotlib to analyze customer purchasing behavior, product performance, sales trends, and country-wise revenue. The repository includes the complete Jupyter Notebook (`EDA.ipynb`) and a detailed project report (`EDA_Documentation.pdf`) documenting the methodology, visualizations, and key business insights.
## Dataset

The dataset used in this project is publicly available and can be downloaded from the UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/502/online+retail+ii
